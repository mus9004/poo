Solución de problemas de conexión de red

Una de las causas más frecuentes de los problemas que ocurren para ver Netflix es una conexión a Internet débil o intermitente. Las repeticiones de almacenamiento en búfer o las cargas frecuentes, los problemas para iniciar la aplicación de Netflix, los mensajes de error que indican que no es posible conectarse a Internet o los problemas para reproducir una película o serie en un dispositivo, generalmente indican que la conexión a Internet es lenta o sufre interrupciones. Estos problemas de streaming vienen acompañados, algunas veces, de códigos de error. Les hemos asignado a estos códigos pasos para la solución de problemas. Si encuentras un código de error, búscalo en nuestro Centro de ayuda.

Nota: Si, después de seguir los pasos que se indican a continuación, todavía tienes problemas para conectarte con el servicio de Netflix, lo recomendable es que te comuniques con tu proveedor de servicios de Internet. Este podrá ayudarte a determinar si el enrutador está configurado adecuadamente para comunicarse con los otros dispositivos de la red doméstica.


Si experimentas problemas de conexión a la red mientras intentas recibir transmisiones desde Netflix, los pasos anteriores son un buen punto de partida. Si ya intentaste todas nuestras sugerencias y sigues teniendo problemas de conexión, es posible que necesites ayuda para solucionar problemas con la red doméstica.
